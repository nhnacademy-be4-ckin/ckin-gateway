# CKIN Gateway Server

## ⛳️ Getting Started

1. Repository 클론

```bash
git clone https://github.com/nhnacademy-be4-ckin/ckin-gateway.git
```

2. ckin-gateway 폴더로 이동

```bash
cd ckin-gateway
```

3. maven project build

```bash
./mvnw clean install
```

4. spring boot 실행

```bash
./mvnw spring-boot:run
```
